// About

Hi, I'm Eugene Melnikov

Character animator since 2006. I lead an animation outsource division, mentoring animators and shaping the creative process from brief to final shot.

Alongside animation, I write Python scripts to improve pipeline workflows and smooth out collaboration between animation teams. My tools are born out of real production pain — if something is repetitive and boring, I automate it.

Building Windows utilities is part hobby, part pragmatism — 50/50. I enjoy thinking up features, optimising processes, and occasionally shipping something useful to the world for free.
