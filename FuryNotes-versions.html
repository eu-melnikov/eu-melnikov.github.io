FuryNotes

FuryNotes is a fast, distraction-free notepad for Windows with one breakthrough idea: there is nothing to save and nowhere to lose. Every note becomes a real file the moment you start typing and saves itself automatically — no Save button, no "where should I put this?" dialog. Your notes stay yours: they'replain Markdown text files you can open anywhere, wrapped in a clean, dark, beautifully formatted editor with folders, links, and instant search across everything.
