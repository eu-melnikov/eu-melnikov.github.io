// About

Excited to share my latest project: RoadMapper! A lightweight desktop tool for visual project planning and team workload management. Built for game developers, animation studios, and creative teams who need a clear view of who's doing what and when. Key features: - Visual timeline with drag & drop tasks - Multi-row tracks for team members - Track collapsing for large projects - Excel import/export - Workdays calculation (excludes weekends) - Color-coded task statuses - Deadline markers - Super Shift mode for batch task moving No cloud. No subscriptions. Just a simple tool that works. Free to download and use.
